# API Contract

This document outlines initial API endpoints planned for the system.

## Auth
- POST /api/auth/login
- POST /api/auth/signup

## Assets
- GET /api/assets
- POST /api/assets
- GET /api/assets/:id
- PUT /api/assets/:id
- DELETE /api/assets/:id

## Dashboard
- GET /api/dashboard

## Bookings
- GET /api/bookings

## Maintenance
- GET /api/maintenance

## Reports
- GET /api/reports

## Audit
- GET /api/audit

## Notifications
- GET /api/notifications
