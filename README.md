# AssetFlow Enterprise Asset & Resource Management System

This workspace now includes a scaffolded structure for:

- Frontend React + Vite app under frontend/
- Backend Express + MongoDB starter under backend/
- Initial components, pages, services, routes, models, controllers, middleware, and utilities

## Quick start

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

## Notes

- The frontend contains placeholder pages and components ready for implementation.
- The backend contains a starter Express server and router/controller/service structure.
- Configure MongoDB in backend/.env if you want the server to connect to a database.
